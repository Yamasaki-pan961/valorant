## 📝 演習問題

### 問題1：攻撃戦術の3段階（基礎）

VALORANTの攻撃戦術は3つの段階に分かれています。Haven攻撃側で残り時間が60秒の時、現在どの段階にいるかを判断し、次に取るべき行動を選んでください。

**現在の状況:**
- A Long: 敵1人確認済み
- B Main: 音なし
- C Long: 不明
- 残り時間: 38秒

- A) Creating Space段階 - さらなる情報収集を継続
- B) Taking Space段階 - エクスキュート戦術の実行
- C) Controlling Space段階 - ポストプラント配置への移行
- D) 時間切れのため強制突撃

<details>
<summary>正解と解説を見る</summary>

**正解: B) Taking Space段階 - エクスキュート戦術の実行**

**解説:**
攻撃戦術の3段階の時間配分：

- **Creating Space（0-30秒）**: 既に完了
- **Taking Space（30-45秒）**: 現在の段階（38秒）
- **Controlling Space（45秒-終了）**: これから準備する段階

**38秒の時点での最適行動:**
- 収集した情報（A Long敵1人）を基に攻撃対象を決定
- B Main音なし = 軽い防御の可能性
- エクスキュート戦術を実行してサイト制圧開始
- 45秒までにスパイク設置を目指す

時間管理は攻撃成功の重要要素で、各段階を適切なタイミングで実行することが勝利への鍵です。
</details>

### 問題2：デフォルト戦術 vs エクスキュート戦術（応用）

Haven攻撃側でチーム構成がSova、Omen、Raze、Skye、Viperです。以下の状況でどの戦術を選択すべきでしょうか？

**状況:**
- 敵チーム: エコラウンド（Specter + Shield程度）
- 自チーム: フルバイ
- 前ラウンドで敵は3-1-1配置（A-B-C）を確認
- 残り時間: 1分40秒

- A) プロアクティブ・デフォルト戦術
- B) リアクティブ・デフォルト戦術  
- C) 即座のエクスキュート戦術
- D) フェイク・エクスキュート戦術

<details>
<summary>正解と解説を見る</summary>

**正解: A) プロアクティブ・デフォルト戦術**

**解説:**
この状況でプロアクティブ・デフォルトが最適な理由：

**敵の状況分析:**
- **エコラウンド**: 軽装備で積極的な行動の可能性
- **3-1-1配置**: 分散配置で各サイトが薄い
- **装備差**: 有利な撃ち合いが期待できる

**プロアクティブ・デフォルトの利点:**
- **敵のユーティリティ消費**: エコラウンドの限られたリソースを早期消費
- **情報収集の積極化**: 軽装備相手に有利な情報戦
- **複数選択肢の創出**: 3サイトのうち最も弱い箇所を特定

**他の選択肢の問題点:**
- B) リアクティブ: エコラウンドの積極性を活用できない
- C) 即座エクスキュート: 情報不足でリスクが高い
- D) フェイク: エコラウンド相手には過度に複雑

経済差がある状況では、積極的なデフォルトで敵のリソースを消費させることが重要です。
</details>

### 問題3：2025年メタ構成の理解（実践）

現在の主流構成「1デュエリスト + 2イニシエーター + 2スモーク」でLotus攻撃を行います。各ロールの適切な配置と役割分担はどれでしょうか？

**チーム構成:** Yoru、Sova、Tejo、Omen、Viper

- A) 全員でA攻撃に集中し、圧倒的な戦力でサイト制圧
- B) 3-1-1配置でデフォルト戦術、情報収集後に集中攻撃
- C) 2-2-1配置で同時攻撃、複数サイトへの分散攻撃
- D) 5人でMid制圧、そこから状況に応じてサイト選択

<details>
<summary>正解と解説を見る</summary>

**正解: B) 3-1-1配置でデフォルト戦術、情報収集後に集中攻撃**

**解説:**
2イニシエーター・2スモーク構成の最適活用法：

**理想的配置（Lotus）:**
- **A Main**: Sova、Yoru、Omen（3人）
- **B Main**: Tejo（1人）  
- **C Main**: Viper（1人）

**各ロールの役割:**
- **Sova**: A Mainでドローン・ダーツによる情報収集
- **Tejo**: B Mainで新エージェントの高性能偵察能力活用
- **Omen**: 全体的なスモーク支援、必要に応じて他サイト移動
- **Viper**: C Mainでウォール使用、Mid分断
- **Yoru**: 情報に基づく最適サイトへのエントリー

**この構成の利点:**
- **情報の多角化**: 異なる手法での同時情報収集
- **柔軟性**: 収集情報に基づく迅速な戦術変更
- **メタ活用**: 現在の主流構成の利点を最大化

**他の選択肢の問題:**
- A) 全員集中: 他サイトが完全に無防備
- C) 分散攻撃: 人数不足で失敗リスク高
- D) Mid制圧: Lotusの構造上、効率が悪い

現在のメタ構成は情報収集力の多角化に重点を置いています。
</details>

### 問題4：トレードキルの実戦判断（高度）

Bind A攻撃でRazeがA Shortからエントリーを試み、敵Cypherに撃ち負けました。この時の最適な対応はどれでしょうか？

**状況:**
- Raze死亡位置: A Short出口付近
- Cypher位置: A Site中央付近（推定）
- 残りチーム: Sova（A Bath）、Omen（A Short後方）、Skye（A Short後方）、Viper（B監視）
- 残り時間: 32秒

- A) 全員でA Shortから一斉突撃してトレード
- B) OmenがA Shortでトレード、他はA Bathに回る
- C) SkyeのフラッシュでOmenがトレード、Sovaは A Bathから挟み撃ち
- D) 一旦撤退してB攻撃に変更

<details>
<summary>正解と解説を見る</summary>

**正解: C) SkyeのフラッシュでOmenがトレード、Sovaは A Bathから挟み撃ち**

**解説:**
効果的なトレードキルの条件を満たした最適解：

**トレードキルの成功条件:**
- **距離**: Omen（A Short後方）→ Cypher（A Site）= 約4m以内 ✓
- **時間**: Raze死亡から3秒以内でのトレード実行可能 ✓  
- **視線**: A Shortから A Siteへの明確な射線あり ✓

**選択Cの戦術的優位性:**
1. **ユーティリティ活用**: Skyeフラッシュで安全にトレード
2. **クロスファイア**: SovaのA Bathからの挟み撃ち
3. **情報活用**: Cypherの位置が推定できている
4. **時間効率**: 32秒で十分な実行時間

**他の選択肢の問題:**
- A) 一斉突撃: 無計画で多数の犠牲者の可能性
- B) 単独トレード: Cypherのトラップやクロスファイアリスク
- D) B変更: 時間不足（32秒）でセットアップ困難

**戦術的価値:**
- 数的均衡の維持（5v5→4v4）
- A Site制圧継続の可能性
- Cypherのユーティリティ消費強制

効果的なトレードキルは単なる復讐ではなく、戦術的優位を維持する重要な行動です。
</details>

### 問題5：適応的攻撃判断（総合）

Haven攻撃側でA攻撃を計画していましたが、以下の情報変化が発生しました。最も適切な戦術変更はどれでしょうか？

**当初の計画:** A攻撃（4人）+ C Lurk（1人）

**情報変化:**
- 25秒時点: C LongでKilljoy確認（防御増強の兆候）
- 28秒時点: A Longでも敵の足音増加
- 30秒時点: B Mainから音なし
- 残り時間: 35秒

- A) 当初の計画通りA攻撃を強行実行
- B) A攻撃を維持、C Lurkを増員してC攻撃に変更
- C) 全員でB攻撃に即座変更  
- D) A Fake実行後、C攻撃に変更

<details>
<summary>正解と解説を見る</summary>

**正解: C) 全員でB攻撃に即座変更**

**解説:**
適応的攻撃の判断基準に基づいた最適解：

**情報分析:**
- **A Site**: 敵増員確認（足音増加）→ Heavy防御
- **C Site**: Killjoy確認 → セットアップ完了の可能性
- **B Site**: 音なし → Light防御の可能性が高い
- **時間**: 35秒 → エクスキュート実行に適切なタイミング

**選択Cの戦術的合理性:**
1. **最弱点攻撃**: 情報から判断される最も防御の薄いサイト
2. **時間効率**: 35秒は新エクスキュートに十分
3. **リスク最小化**: Heavy防御への突撃回避
4. **全戦力活用**: 分散せずに圧倒的優位を作る

**適応的攻撃の原則:**
- **情報優先**: 収集した情報を最大限活用
- **フレキシビリティ**: 計画にこだわらない柔軟性
- **時間管理**: 変更実行の十分な時間確保

**他の選択肢の問題:**
- A) 強行: 情報無視でリスク最大
- B) C変更: Killjoyセットアップ済みで困難
- D) フェイク: 35秒では複雑すぎて時間不足

**成功のポイント:**
- 情報収集プロセスの価値を最大化
- 敵の予想を上回る迅速な戦術変更
- チーム全体での統一された行動

現代VALORANTでは、情報に基づく適応的な戦術変更能力が勝敗を分けます。
</details>

### 問題6：ポストプラント戦術（実戦応用）

Bind A攻撃でスパイクの設置に成功しました。設置位置はA Site中央、残り時間30秒です。以下の配置のうち、最も効果的なポストプラント配置はどれでしょうか？

**チーム状況:** 
- 生存者: Jett、Sova、Omen、Brimstone（4人）
- 敵チーム: 5人生存
- スパイク設置位置: A Site中央（Truck側）

**配置選択肢:**

- A) 全員A Site内でスパイク周辺防御
- B) Jett（A Site）、Sova（A Short）、Omen（A Bath）、Brimstone（Spawn側）
- C) 2人A Site、2人A Short、B方向は放置
- D) Jett（A Site）、他3人はB Teleporter近くで待機

<details>
<summary>正解と解説を見る</summary>

**正解: B) Jett（A Site）、Sova（A Short）、Omen（A Bath）、Brimstone（Spawn側）**

**解説:**
効果的なポストプラント配置の4つの要素を満たした最適解：

**各ポジションの戦術的意味:**

**Jett（A Site内）:**
- **スパイク直接防御**: 解除開始の即座発見
- **機動力活用**: ダッシュでの緊急離脱可能
- **最後の砦**: リテイク失敗時の最終防御線

**Sova（A Short）:**
- **メインルート監視**: A Shortからのリテイク封じ
- **情報収集**: ドローン・ダーツでの敵位置確認
- **遠距離優位**: A Shortの長い射線活用

**Omen（A Bath）:**
- **フランク防止**: A Bathからの奇襲防止
- **スモーク支援**: 必要に応じた視線遮断
- **テレポート活用**: 状況に応じた位置変更

**Brimstone（Spawn側）:**
- **後方安全確保**: Teleporter監視
- **スティム支援**: リテイク阻止の追加火力
- **ウルト準備**: スパイク防御の最終手段

**この配置の利点:**
1. **360度カバー**: 全方向からのリテイクに対応
2. **クロスファイア**: 複数角度からの攻撃可能
3. **情報収集**: Sovaスキルでの継続監視
4. **退却路確保**: 各ポジションから安全な撤退可能

**他の選択肢の問題:**
- A) 全員サイト内: フランクに無防備、逃げ場なし
- C) B方向放置: Teleporter奇襲の危険性
- D) B待機: A Site放棄でスパイク解除許可

**時間管理の重要性:**
- 30秒 = スパイク爆破まで十分な時間
- 解除には7秒必要 → 23秒まで防御すれば勝利
- 各ポジションでの時間稼ぎが勝利の鍵

効果的なポストプラント配置は、単なる待機ではなく積極的な防御戦術です。
</details>

# 第2章 復習・理解度確認問題集

## 📋 演習の目的

第2章で学んだ内容の理解度を確認し、エリア理論の基本概念を定着させることを目的とします：

- エリアコントロールの重要性とその3つの状態
- マップ構造の理解（サイト、接続エリア、進入ルート）
- 空間支配と勝利条件の関係
- 攻撃・防御における具体的なエリア戦略
- 段階的エリア拡張の原則と実践

## 💪 基本概念の理解度確認

### 問題1：エリアコントロールの本質

**第2章で学んだ内容の確認**

**問題**: エリアコントロールの正しい定義はどれか？

**選択肢**:

- A) 特定のエリアに留まること
- B) 特定のエリアに対する安全なアクセス権を確保すること
- C) 特定のエリアで敵を倒すこと
- D) 特定のエリアを素早く通過すること

<details>
<summary>正解と解説を見る</summary>

**正解**: B) 特定のエリアに対する安全なアクセス権を確保すること

**解説**: 第2章で学んだ通り、エリアコントロールとは特定のエリアに対する「安全なアクセス権」を確保することです。単に留まることではなく、自由に利用できる権利を得ることが重要です。

</details>

### 問題2：エリアコントロールの3つの状態

**第2章で学んだ内容の確認**

**問題**: エリアコントロールの3つの状態として正しいものはどれか？

**選択肢**:

- A) 攻撃・防御・中立
- B) 完全コントロール・部分コントロール・敵コントロール
- C) 前進・維持・後退
- D) 安全・危険・不明

<details>
<summary>正解と解説を見る</summary>

**正解**: B) 完全コントロール・部分コントロール・敵コントロール

**解説**: 第2章で学んだエリアコントロールの3つの状態です。完全コントロール（安全に使用可能）、部分コントロール（条件付きで使用可能）、敵コントロール（敵が支配）の3つです。

</details>

### 問題3：マップ構造の理解

**第2章で学んだ内容の確認**

**問題**: VALORANTマップの構成要素として正しいものはどれか？

**選択肢**:

- A) サイト・接続エリア・進入ルート
- B) 前線・中線・後方
- C) 上・中・下
- D) 左・右・中央

<details>
<summary>正解と解説を見る</summary>

**正解**: A) サイト・接続エリア・進入ルート

**解説**: 第2章で学んだマップ構造の3つの要素です。サイト（スパイク設置可能エリア）、接続エリア（異なるエリアを結ぶ通路）、進入ルート（サイトへの侵入経路）で構成されています。

</details>

### 問題4：攻撃側のエリア戦略

**第2章で学んだ内容の確認**

**問題**: 攻撃側の段階的エリア拡張の正しい順序はどれか？

**選択肢**:

- A) サイト攻撃 → 情報収集 → 初期コントロール → エリア拡張
- B) 初期コントロール → 情報収集 → エリア拡張 → サイト攻撃
- C) 情報収集 → 初期コントロール → エリア拡張 → サイト攻撃
- D) エリア拡張 → サイト攻撃 → 情報収集 → 初期コントロール

<details>
<summary>正解と解説を見る</summary>

**正解**: B) 初期コントロール → 情報収集 → エリア拡張 → サイト攻撃

**解説**: 第2章で学んだ段階的エリア拡張の正しい順序です。まず安全なエリアを確保し、そこから情報を収集、段階的に支配エリアを拡大し、最終的にサイト攻撃を行います。

</details>

### 問題5：防御側のエリア戦略

**第2章で学んだ内容の確認**

**問題**: 防御側の階層的防御の正しい順序はどれか？

**選択肢**:

- A) リテイク → サイト防御 → 中間防御 → 外周防御
- B) 外周防御 → 中間防御 → サイト防御 → リテイク
- C) サイト防御 → 外周防御 → 中間防御 → リテイク
- D) 中間防御 → 外周防御 → リテイク → サイト防御

<details>
<summary>正解と解説を見る</summary>

**正解**: B) 外周防御 → 中間防御 → サイト防御 → リテイク

**解説**: 第2章で学んだ階層的防御の順序です。外周で早期発見、中間で進行遅延、サイトで最終阻止、リテイクで奪還という流れです。

</details>

### 問題6：エリアの戦略的価値

- C) そのエリアが勝利条件に与える影響

<details>
<summary>正解と解説を見る</summary>

</details>

## 🎮 実戦例の理解度確認

### 問題7：Haven Mid制圧の戦略的意義

**第2章の状況例「Haven Mid エリアコントロール」について**

**問題**: Haven Mid Windowを制圧する戦略的価値として正しいものはどれか？

**選択肢**:

- A) 単に中央を取るため
- B) 3サイトへの攻撃選択肢を得るため
- C) 敵を混乱させるため
- D) 時間を稼ぐため

<details>
<summary>正解と解説を見る</summary>

**正解**: B) 3サイトへの攻撃選択肢を得るため

**解説**: 第2章の状況例で学んだ通り、Mid Windowを制圧することでA Short、B Connector、C Shortの3つのルートから攻撃選択肢を得ることができます。これにより敵は3サイトすべてを守る必要が生じ、攻撃側が有利になります。

</details>

### 問題8：段階的エリア拡張の実践

**第2章の実戦例「段階的エリア拡張の実践」について**

**問題**: 段階的エリア拡張で最も重要なポイントは何か？

**選択肢**:

- A) 素早く次のエリアに移動する
- B) 各段階で確実にエリアを制圧してから次に進む
- C) 複数のエリアを同時に攻撃する
- D) 敵を発見したら即座に撤退する

<details>
<summary>正解と解説を見る</summary>

**正解**: B) 各段階で確実にエリアを制圧してから次に進む

**解説**: 第2章の実戦例で学んだ重要なポイントです。盲目的な進行を避け、各段階で確実にエリアを制圧することで、安全で効果的なエリア拡張が可能になります。

</details>

### 問題9：複合的エリア戦略の判断

**第2章の実戦例「Mid制圧による3サイト攻撃選択肢」について**

**問題**: A Siteにオペレーター、B Siteに敵の存在確認、C Siteの状況不明の場合、最適な判断は何か？

**選択肢**:

- A) A Site攻撃を強行する
- B) B Site攻撃を選択する
- C) C Site攻撃を選択する
- D) 攻撃を諦めて次ラウンドに備える

<details>
<summary>正解と解説を見る</summary>

**正解**: C) C Site攻撃を選択する

**解説**: 第2章の実戦例で学んだ判断基準です。A Siteの強固な守備（オペレーター）とB Siteの確認済みの守備を避け、守備状況が不明で薄い可能性があるC Siteを攻撃することが最適です。

</details>

## 📊 実戦シミュレーション問題

### 問題13：複合判断問題

**状況設定**:
- **マップ**: Haven攻撃側
- **時間**: 残り50秒
- **人数**: 4v5（1人劣勢）
- **現在位置**: A Long制圧済み、Mid未制圧、C Long情報なし

**問題**: この状況で最適な戦略は何か？

**選択肢**:

- A) A Long から A Site直接攻撃
- B) Mid制圧を試みる
- C) 全員でC Long に移動
- D) A Long から A Site情報収集後、状況に応じて判断

<details>
<summary>正解と解説を見る</summary>

**正解**: D) A Long から A Site情報収集後、状況に応じて判断

**解説**: 段階的エリア拡張の原則に従い、制圧済みのA Longから次のエリア（A Site）の情報を収集することが最適です。人数劣勢と時間制約を考慮し、情報に基づいて最終判断を行います。

</details>

### 問題14：防御側エリア戦略

**状況設定**:
- **マップ**: Haven防御側
- **時間**: 残り1分20秒
- **人数**: 5v5
- **敵情報**: A Long方向で音確認、B・C側情報なし

**問題**: この状況での最適な防御配置は何か？

**選択肢**:

- A) 全員でA Site防御
- B) 2人A Site、2人B Site、1人C Site の配置維持
- C) 1人A Site、2人B Site、2人C Site に移動
- D) 1人A Site、4人B・C Site に移動

<details>
<summary>正解と解説を見る</summary>

**正解**: B) 2人A Site、2人B Site、1人C Site の配置維持

**解説**: 敵の動きが確認されたA Siteを警戒しつつ、B・Cサイトも最低限の防御を維持する配置が最適です。Havenは3サイトマップなので、バランスよく配置することが重要です。

</details>

### 問題15：エリア価値の総合判断

**状況設定**:
- **マップ**: Haven攻撃側
- **時間**: 残り1分30秒
- **人数**: 5v5
- **現在**: 各エリアの情報なし

**問題**: 最初に制圧すべきエリアとして最も価値が高いものはどれか？

**選択肢**:

- A) A Long
- B) C Long
- C) Mid Window
- D) A Short

<details>
<summary>正解と解説を見る</summary>

**正解**: C) Mid Window

**解説**: 第2章で学んだエリアの戦略的価値に基づき、Mid Windowは3サイトへの攻撃選択肢を提供する最も価値の高いエリアです。A Long、C Longは単一サイトへのアクセスのみを提供します。

</details>
